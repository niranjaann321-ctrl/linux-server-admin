# Linux Server Administration Project

## Project Overview

This project demonstrates the installation, configuration, security hardening, monitoring, performance evaluation, and security auditing of an Ubuntu Linux server environment using VirtualBox.

The project was completed across multiple phases covering:

- Ubuntu Server deployment
- SSH remote administration
- Monitoring and testing tools
- Performance benchmarking
- Firewall and SSH hardening
- Security automation
- System auditing and vulnerability assessment

---

# Environment

## Virtualization Platform

- Oracle VirtualBox

## Operating System

- Ubuntu Server 26.04 LTS

## Network Configuration

- SSH Remote Access
- Host-Only Networking
